  <behavior>
    - get() returns undefined for a key that was never set
    - get() returns the cached value for a key that was set and has not expired
    - get() returns undefined for a key whose TTL has expired
    - set() stores a value retrievable by get()
    - has() returns true for existing non-expired keys, false otherwise
    - Cache automatically cleans up expired entries periodically (every 60s)
    - destroy() stops the cleanup interval
    - Cache key is built from: CPF (cleaned digits) + providerId + searchType
  </behavior>
  <action>
    Create server/services/consultation-cache.service.ts with:

    1. A generic Map-based TTL cache class `TtlCache<T>`:
       - constructor(ttlMs: number, cleanupIntervalMs: number = 60_000)
       - Internal Map<string, { value: T; expiresAt: number }>
       - get(key: string): T | undefined — returns value if not expired, undefined otherwise
       - set(key: string, value: T): void — stores with expiresAt = Date.now() + ttlMs
       - has(key: string): boolean — true if key exists and not expired
       - delete(key: string): void
       - clear(): void
       - size: number (getter, counts only non-expired)
       - destroy(): void — clears the cleanup interval
       - Private _cleanup() method that runs on setInterval to purge expired entries

    2. A specific `ConsultationCache` class that wraps TtlCache:
       - TTL default: 5 minutes (300_000 ms), configurable via constructor
       - buildKey(cpf: string, providerId: number, searchType: string): string
         Returns `${cpf.replace(/\D/g, '')}:${providerId}:${searchType}`
       - getResult(cpf: string, providerId: number, searchType: string): CachedResult | undefined
       - setResult(cpf: string, providerId: number, searchType: string, result: CachedResult): void
       - hasResult(cpf: string, providerId: number, searchType: string): boolean

    3. CachedResult interface:
       ```typescript
       export interface CachedResult {
         result: any;  // The full consultation result object
         consultation: any;  // The saved consultation record
         cachedAt: number;  // Date.now() when cached
       }
       ```

    4. Export a singleton: `export const consultationCache = new ConsultationCache();`

    IMPORTANT: Do NOT use any external library. Pure Map-based implementation.
    IMPORTANT: The cleanup interval must use unref() if available (Node.js) to not prevent process exit.
  </action>
